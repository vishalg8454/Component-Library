Badge

Badge can be used to display notification count or a status such as offline or online. Badge is available in four sizes 'xsm', 'sm', 'md' and 'lg'. The three status modes are badge-online, badge-offline and badge-away.

Button

Primary and secondary buttons are available in two varients: Solid and Outline. Link buttons are available under class btn-link. FAB are available under class button-floating-action. Icons can also be used inside button.

Card

Card is a major component which can be used to display information about something like products. They can also be used to confim user actions. Horizontal and vertical card are available with pictures in them. Text-only cards with buttons are also possible.

Avatar

Avatar is used to show profile picture or initials of an user. Avatar is available in four sizes 'xsm', 'sm', 'md' and 'lg'.

Four sizes of text Avatar are available in sizes avatar-xsm, avatar-sm, avatar-md and avatar-lg.

Image

Image can be responsive where can grow in size while maintaining aspect ratio. Round images are available under the class img-round.

Input

Input element with type="text" can be used to take textual inputs from user. The styled input box is available under input-box class along with the styling applied for invalid validations. The label styling is available under input-label.

Rating

Rating is used to take rating feedback from the user. Rating icons base styling is available under rating-icon. The checked status can be applied using checked class.

Text-Utility

Text Utility classes are available for styling text. Header's size can be chosen using classes from text-h1 through text-h6. Text size can be increased or decreased under class name text-lg, text-md and text-sm. Primary color and gray color available under text-primary and text-gray.
